## 2 蓝莺IM服务端API文档

### 说明

> 调用蓝莺IM API前，请先从控制台获取参数api_endpoint、app_id、access-token。api_endpoint是服务地址，app_id，access-token在请求Header中使用。未特殊说明的请求Content-Type类型为application/json。
